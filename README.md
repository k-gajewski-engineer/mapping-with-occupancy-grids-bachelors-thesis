# Bachelor's Thesis Results

This repository contains selected materials related to my bachelor's thesis on localization and mapping using occupancy grids. It includes the thesis defense presentation and the experimental results obtained in both simulation and real-world tests.

## Project Overview

The thesis focused on developing a localization and occupancy grid mapping algorithm for a mobile robot operating in a 2D environment. The solution was implemented and evaluated in the MATLAB environment using both simulated data and experiments conducted on the Adept MobileRobots Pioneer 3-AT platform.

## Repository Contents

This repository includes:
- the thesis defense presentation
- selected figures and experimental results
- maps generated during simulation and real-world experiments

## Experimental Scope

The algorithm was tested in two environments:
- simulation
- real robot experiments

The evaluation focused on the quality of the generated occupancy maps and the behavior of the localization process under practical operating conditions.

## Results

The repository presents the final outcomes of the project, including visualizations of generated maps and selected comparison results. These materials summarize the main conclusions of the thesis and illustrate the performance of the proposed approach.

## Source Code Notice

The source code is not included in this repository. Only presentation materials and selected experimental results are shared publicly.
